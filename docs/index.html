What This Page Shows

  • This viewer replays exported snapshots from the Python model rather than rerunning the simulation in the browser.
  • The replay is sampled every fixed number of simulation steps to keep the static data bundle small enough for GitHub Pages.
  • The charts use the full per-step histories from the export summary, while the world canvas uses the sampled frames.
